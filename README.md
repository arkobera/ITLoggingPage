# ITLoggingPage

## Project Description

This project is a web application designed for logging and visualizing IT-related issues. It allows users to report problems, track their status, and view a history of logged issues.

## Features

*   Log new IT issues with details such as title, description, severity, and status.
*   View a list of all logged issues.
*   Filter and sort issues based on status, severity, or date.
*   Update the status of existing issues.

## Steps to Reproduce

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/ITLoggingPage.git
    cd ITLoggingPage
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the application:**
    ```bash
    npm start
    ```

4.  **Access the application:**
    Open your web browser and navigate to `http://localhost:3000` (or the port specified in your configuration).
